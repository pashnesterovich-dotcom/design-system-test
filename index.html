How to Use

Integration
⚡
Lovable / v0
Paste index.css in your project's global styles. Reference variables directly: var(--bg-surface), var(--accent-primary). All component classes are ready to use.
⚛️
React + Tailwind
Import tailwind.config.ts and extend your theme. Use token names as Tailwind classes: bg-surface, text-secondary, border-default. All spacing and radius values map directly.
🎨
JSON Import
Use tokens.json with Style Dictionary v4. Schema is SD-compatible. Run npx style-dictionary build to regenerate CSS, Tailwind, or any other format from the single source of truth.
